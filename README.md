# AI-Powered Travel Recommendation Platform

The capstone of the journey. Bring every rung together into a real platform: shared config and a Postgres catalog; destination and stay search with full-text and tag filters; a personalized ranking engine that blends weighted features with each traveler's interaction history; Redis-cached personalization profiles for fast, scalable serving; LLM personalized suggestions and grounded itinerary generation with OpenAI; resilient generation with retries and graceful degradation; a React frontend with listings, a map, and a trip plan; booking-style API workflows; a recommendation-quality feedback loop with CTR and booking-rate dashboards; full observability with Prometheus/Grafana metrics for engagement, ranking quality, and LLM cost/latency; and a one-command Docker Compose deployment ready for AWS. Optimized for fast, scalable personalization.

Built step-by-step with [KhwajaLabs Build](https://khwajalabs.com).

- Python
- FastAPI
